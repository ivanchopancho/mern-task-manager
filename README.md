## Task Manager (MERN)

A full-stack task manager built with the MERN stack.  
Users can register, log in, and manage their own tasks securely.

### Features
- JWT authentication
- User-scoped tasks
- Create, update, delete, and complete tasks
- Optional task descriptions
- Responsive UI with Tailwind CSS

### Tech Stack
- React
- Node.js / Express
- MongoDB / Mongoose
- JWT Authentication
